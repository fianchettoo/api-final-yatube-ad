
# API для Yatube

REST API для платформы для ведения блогов Yatube.

Проект предоставляет программный доступ к основным сущностям Yatube и возвращает данные в формате JSON. Аутентификация выполняется с помощью JWT-токенов.

## URL

- API: `http://localhost:8000/api/v1/`
- Документация: `http://localhost:8000/redoc/`

## Ресурсы

- `posts` — создание, просмотр, обновление и удаление постов
- `comments` — создание, просмотр, обновление и удаление комментариев к постам
- `follow` — просмотр подписок и подписка на авторов
- `groups` — просмотр групп
- `auth` — получение и обновление JWT-токенов


## Локальная установка

1. Склонируйте/скачайте репозиторий.

2. Установите виртуальное окружение:

```bash
python -m venv venv

. venv/Scripts/activate
```

3. Установите зависимости из файла `requirements.txt`:

```bash
pip install --upgrade pip

pip install -r requirements.txt
```

4. Примените миграции:

```bash
python manage.py migrate
```

3. Запустите сервер разработки из корневой директории проекта:

```bash
python manage.py runserver
```
